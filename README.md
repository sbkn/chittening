##

### Scripts

* Run dev server on localhost:3000:

    `npm start`


* Run tests:

    `npm test`


* Run tests and watch:

    `npm run test:watch`


* Run tests and generate code coverage:

    `npm run test:cov`


* Run eslint:

    `npm run lint`


* Generate eslint report (saved to eslint-report.html):

    `npm run lint:report`
